# Todo-list-vue

仿照[Todos App](http://todos.netcore.io/)通过 Vue 完成了 todo-list，细颗粒度组件，组件之间使用`eventbus`实现通讯;

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
